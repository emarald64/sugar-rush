# Sugar Rush

A 2D platformer inspired by Celeste with a speed boosting powerup that kills you after 4 seconds. 

## Controls

a/left arrow: move left
d/right arrow: move left
r: respawn
esc: pause
F3: toggle noclip, if you are testing (and bad at videogames). noclip scores won't count for total scores
